# TradeCapture (TP ICAP) Demo/Test solution

### Requirements 
- Single buy
- Thread safe
- Event driven
- Fully testable
- Stop after an error

### Projects
- TradeCapture.Foundation - Foundataion classes (Events, Notifications, Interfaces)
- TradeCapture.Core       - Order implementation
- TradeCapture.Tests      - Unit tests  

### Build
- dotnet build
- dotnet test
